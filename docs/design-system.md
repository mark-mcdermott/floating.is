# floating.is — Design System

The visual identity of floating.is should feel:

- quiet
- premium
- editorial
- restrained
- architectural
- atmospheric
- tactile
- intentional

The design language should resemble:
- an independent software studio
- a modern architecture magazine
- a calm gallery catalog
- software built slowly and carefully

Avoid:
- loud SaaS aesthetics
- Product Hunt energy
- neon gradients
- generic startup UI
- glassmorphism
- oversized shadows
- clutter
- trendy Dribbble effects

Whitespace is a core design element.

---

# Color Palette

## Light Mode

Backgrounds:
- #f6f3ef
- #f4f1ed
- #efebe7

Primary text:
- #161616
- #1e1e1e

Secondary text:
- #7a746d
- #8a847c

Borders:
- border-black/10
- border-black/5

Accent:
- #97a8bb
- #9aa9b8

Never use saturated colors.

---

## Dark Mode

Backgrounds:
- #06070a
- #0b0d11
- #0e1015

Primary text:
- #f2eee8
- #f6f2ed

Borders:
- border-white/10

Accent:
- muted dusty blue-gray

Dark mode should feel:
- cinematic
- nocturnal
- quiet
- elegant

Never cyberpunk.

---

# Typography

Typography drives the aesthetic.

## Serif Direction

Examples:
- Canela
- Editorial New
- Noe Display
- Austin
- Bodoni
- Didot
- Playfair Display

Use serif for:
- page titles
- hero typography
- article titles
- emotional emphasis

Headlines should feel:
- oversized
- calm
- luxurious
- spacious

---

## Sans Serif Direction

Examples:
- Inter
- Suisse
- Söhne
- Neue Haas Grotesk
- SF Pro

Use sans-serif for:
- body text
- navigation
- forms
- labels
- metadata

---

# Layout Principles

Prefer:
- asymmetrical balance
- generous whitespace
- constrained widths
- modular composition
- soft hierarchy

Typical containers:
- `max-w-7xl`
- `mx-auto`
- `px-6 md:px-8`

Spacing should feel slow and breathable.

Avoid dense layouts.

---

# Shadows

Shadows should be extremely restrained.

Light mode:
- soft ambient separation only

Dark mode:
- gentle glow separation

Never:
- heavy floating cards
- dramatic blur effects
- aggressive elevation

---

# Navigation

Navigation should feel:
- tiny
- elegant
- understated
- editorial

Typical navigation:
- Work
- Studio
- Journal
- Contact

Current page:
- thin underline
- subtle emphasis

---

# Footer

Footer should feel:
- architectural
- quiet
- restrained

Include:
- studio descriptor
- location
- navigation
- subtle ambient toggle

---

# Ambient Circle

The small ambient circle element should feel:
- mysterious
- calm
- premium
- interactive

Use:
- thin borders
- tiny centered dot
- subtle hover states

Never flashy.
